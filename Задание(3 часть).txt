Работа по вёрстке страницы:
1) Приводим внешне страницы к симпатичному виду:
    1) Для <section> добавим стили:
        Максимальная Ширина 1240px
        Внешние отступы "0 auto" - позволяет центрировать элемент, если он не на всю ширину
        Внутрение отступы "10px 30px"
    2) Для <body> добавим стили
        Цвет фона (лучше если он будет, даже если и белый)
        Минимальная высота 100vh
        Возможно, отобразите как flex контейнер, колонкой, с отступами между дочерними элементами, чтобы занимали всю высоту
    ////Сделано
    3) Добавляем цвета, стили и поведение для всех элементов на странице, чтобы выглядело хорошо:
        Для ссылок стили "обычные", при "наведении мышки" и в "активном" состоянии
        Для tab ссылок стили обычные и в активном состоянии
        Цвета для текстов, фон страницы, header и footer элементов
        Закрывать мобильное меню, когда мы кликаем по ссылкам внутри него

        2*) Если есть желание - добавить проверку, если браузер в тёмном режиме, чтобы сделать ночную версию сайта, чтобы поберечь глаза пользователя
            В JS добавить проверку тёмного режима
            const isDarkMode = () => 
  window.matchMedia && window.matchMedia('(prefers-color-scheme: dark)').matches;
            В таком варианте на <html> или <body> добавить класс "dark-mode"
            В CSS добавить для этого класса иные цвета сайта, чтобы были темнее и легче для глаза цвета в ночное время

    4) Для раздела "Наши работы" или его аналога - добавить Owl Slider
        1) Слайдер, где будет карусель, с примерами работ, с картинками (если нужно) и текстовым описанием к ним
        (пример фотографий на вашу тематику)
        2) Добавить свойство зацикленности, чтобы они по кругу повторялись
        3) Добавить для него параметр автопрокрутки, чтобы слайды по таймеру каждые несколько секунд менялись автоматически
        4) Уберите точки, добавьте стрелки
        5) Добавить параметр, чтобы автопрокрутка ставилась на паузу, при наведении мышки
        6*) Замените в CSS стили для стрелок, чтобы они были с помощью иконок, которые вы выбрали раньше, при подготовке страницы

    5) Для главной страницы создайте структуру, по примеру "Пример.jpg", находящемся в этой папке
        1) С помощью CSS Grid создайте HTML структуру подходящей формы
        2*) Для мобильного отключать Grid, чтобы они выводились друг под другом, либо каруселью

(На ваше усмотрение, не обязательно)
    6*) В раздел "Контакты" или его аналог добавим Карту. На ваш выбор, Гугл или Яндекс
	Если Гугл - урок по ссылке:
		https://developers.google.com/maps/documentation/javascript/adding-a-google-map?hl=ru
	Если Яндекс:
        1) По ссылке https://yandex.ru/maps-api/products/js-api#price
            Можно выбрать бесплатный вариант (ограничение до 1000 запросов сервису в сутки - более чем достаточно для пет проекта или для портфолио)
        2) Добавить на страницу JS и HTML код
            (Инструкция https://yandex.ru/dev/jsapi30/doc/ru/quickstart)
            Место, где подключаем JS - когда видите кусок "YOUR_API_KEY" - в этом месте вставляете свой ключ API, который получите после выполнения пункта выше

        JS код для инициализации карты лучше вынесите в отдельный файл, не в script.js, будем разграничивать разный функционал по отдельным файлам, чтобы не создавать нечитаемую свалку код
